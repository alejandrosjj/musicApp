**BACKEND**

**Install**

npm install

npm install express cors morgan sequelize cookie-parser mysql2

npm install sequelize-cli (create tables, databases...)


**Create DataBase**

npx sequelize-cli db:create


**Create Table**

npx sequelize-cli db:migrate

**Insert**

npx sequelize-cli db:seed:all


npm run start
